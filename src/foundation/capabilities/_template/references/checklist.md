# Capability Checklist

- Mission is narrow and professionally actionable.
- Selection rules distinguish this capability from adjacent entries.
- Risk escalation rules name concrete thresholds.
- Critical details capture expert misses, not generic advice.
- Output contract is structured enough for skill compilation.
- Quality gate is verifiable by a reviewer.
- Used-by list matches registry intent.
- Handoff target is explicit when the capability cannot decide alone.
