# Backup Recovery Checklist

- Identify protected data, metadata, files, indexes, keys, and configuration.
- Define failure scenarios, backup source, retention, encryption, and access controls.
- State RPO, RTO, restore target, owner, and escalation path.
- Validate restore before relying on backup for release safety.
- Check application compatibility after restore.
- Document irreversible operations, residual risk, and compensating recovery options.
