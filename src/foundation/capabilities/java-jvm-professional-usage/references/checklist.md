# Java JVM Professional Usage Checklist

- Verify transaction boundaries and proxy behavior.
- Review thread pools, queues, blocking IO, and naming.
- Separate business, system, and external dependency exceptions.
- Check type modeling and DI lifecycle boundaries.
- Review ORM leakage and pass-through service risks.
- Assess allocation and GC behavior on hot paths.
- Add integration or transaction tests where needed.
- Confirm observability and operational ownership.
