# Language Runtime Selection Checklist

- Classify workload shape and latency, throughput, memory, and deployment constraints.
- Compare type safety, concurrency, async, exception, GC, and memory behavior.
- Review team skill, operational ownership, and incident response readiness.
- Check package ecosystem maturity, security posture, and release lifecycle.
- Define deployment artifact shape and observability tooling.
- Identify interoperability, migration, and rollback costs.
- Select matching language-specific usage capability.
- Record explicit risk acceptance for novelty.
