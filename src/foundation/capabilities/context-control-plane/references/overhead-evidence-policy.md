# Overhead Evidence Policy

Any claim that context control improves routing, professionalism, validation quality, or efficiency needs overhead evidence.

## Required Fields

- selected capability count
- selected reference count
- skipped reference count
- budget mode
- token estimate or `not_collected`
- turn count or `not_collected`
- benchmark case or fixture id
- over-routing guard
- under-routing guard
- caveat describing structural-only or non-live evidence

Do not run live Codex or agent-behavior benchmarks unless an explicit maintainer flag authorizes them. Structural route and fixture validation may run without live model calls.
