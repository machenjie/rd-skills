# Technology Stack Selection Checklist

- State the product constraints, scale assumptions, team maturity, and operating model.
- Compare boring proven options before accepting novelty.
- Review security posture, dependency surface, license fit, and ecosystem maturity.
- Estimate maintenance, hiring, upgrade, hosting, and migration cost.
- Identify integration constraints and unsupported requirements.
- Classify reversibility and exit path.
- Name owners for operation, upgrades, and incident response.
- Record rejected alternatives with concrete tradeoffs.
