# Security And Privacy Checklist

- Verify authentication and session assumptions.
- Verify object-level authorization, tenancy, role, and ownership checks.
- Validate inputs and encode outputs.
- Check XSS, CSRF, SSRF, SQL injection, command injection, RCE, and unsafe deserialization.
- Check secrets, credentials, token scope, rotation, and logging.
- Review dependency, supply-chain, and license risk.
- Review privacy data minimization, retention, consent, access, and deletion.
- Review AI prompt injection, tool permissions, data exposure, and untrusted output handling.
- Review Web3 key custody, signing, transaction replay, and asset movement where relevant.
