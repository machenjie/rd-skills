# Experience Checklist

- Define primary and secondary user flows.
- Identify entry points and exits.
- Map page, modal, drawer, and navigation transitions.
- Specify empty, loading, error, success, disabled, and partial states.
- Define form validation timing and messages.
- Check keyboard access, focus order, labels, announcements, and contrast.
- Review responsive layout and text overflow risk.
- Identify analytics or event tracking needs.
- Define manual and automated UX verification.
